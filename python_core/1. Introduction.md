# Introduction

Python is:

- procedural, statement-based language,
- **dynamically typed** — python keeps track of types for you automatically instead of requiring declaration code,
- **strongly typed** — you can perform on an object only operations that are valid for its type.

#### Conceptual Hierarchy

Python programs may be decomposed into:

- **Programs** are composed of **modules**
  - **Modules** contain **statements**
    - **Statements** contain **expressions**
      - **Expressions** create and process **objects**
        - Objects
          - variables
          - values

Everything is an object in Python.

Python code consist of:

- Defenitions (?):

  - Function defenition (statement / expression)

    ```python
    def func():
        pass
    ```

  - Class defenition (statement / expression)

- Calls:

  - Function call
  - Class call (?)

- Statements:

  - pass
  - assignment
  - IF / ELSE statement
  - WHILE loop statement
  - FOR loop statement

- Expressions:

  - IF / ELSE expressions
  - Comprehensive expressions
  - Get attribute expression

- Operators:

  - Create lambda function operator:

    ```python
    lambda args: expression
    ```

  - Get attribute operator:

    - obj.attr

  - Ariphmetic operations:

    ```python
    x + y # Addition
    x - y # Substraction
    x * y # Multiplication
    x / y # Division (true)
    x // y # Division (floor)
    
    # **Attention!** Floor division (//) rounds result to the less nearest ceil
    
    x % y
    x ** y
    
    # Functional operations
    abs(-3.0) # 3.0
    round(2.3264, 2) # 2.33
    pow(3,4) # 81
    ```

    Negation:

    ```python
    -x
    +x
    ```

  - Logical operators:

    ```python
    x if y else z
    x or y
    x and y
    not x
    ```

  - Membership and class operators:

    ```python
    x in y
    x not in y
    
    x is y
    x is not y
    ```

  - Comparision:

    ```python
    # If bigger or smaller
    
    x < y
    x > y
    x <= y
    x >= y
    
    # If equal or not
    
    x == y
    x != y
    ```

  - Bitwice operations:

    ```python
    x | y
    x ^ y
    x & y
    x << y
    x >> y
    ~x
    ```

  - 

  - Indexing:

    ```python
    x[i]
    x[i:j:k]
    ```


Syntax rules:

- Indention
- Multiline code
- Variable declaration:
  - Python variables never need to be declared ahead of time. A variable is created when you assign it a value.
  - You don't have to explicitly designate variable type when assigning.
