### 1.1. Decimal

Fixed-precision number object.

#### 1.1.1. Create a decimal number

Ordinary floating-point number:

```python
f = 0.1 + 0.1 + 0.1 - 0.3

print(f)

> 5.551115123125783e-17
```

Decimal:

```python
from decimal import Decimal

d = Decimal('0.1') + Decimal('0.1') + Decimal('0.1') - Decimal('0.3')

print(d)

> Decimal('0.0')

d = Decimal('0.1') + Decimal('0.10') + Decimal('0.10') - Decimal('0.30')

print(d)

> Decimal('0.00')
```

#### 1.1.2. How decimal's print operation works, give example?

```python
from decimal import Decimal

print(Decimal(1) / Decimal(7))

> 0.1428571428571428571428571429

x = Decimal(1) / Decimal(7)

x

> Decimal('0.1428571428571428571428571429')
```

#### 1.1.3. How to set decimal precision globally?

```python
import decimal

decimal.getcontext().prec = 4

x = decimal.Decimal(1) / decimal.Decimal(7)
x

> Decimal('0.1429')

print(x)

> 0.1429
```

#### 1.1.4. How to set decimal precision locally?

```python
import decimal

with decimal.localcontext() as ctx:
    ctx.prec = 2
    x = decimal.Decimal('1.00') / decimal.Decimal('3.00')
    print(x)

> 0.33

y = decimal.Decimal('1.00') / decimal.Decimal('3.00')
print(y)

> 0.3333333333333333333333333333
```

### 1.2. Fractions (rationals)

Frational number object with numerator and denominator.

#### 1.2.1. Create a fraction number (from integers and float) and show how print function overloaded for fractions

From two integers:

```python
from fractions import Fraction

x = Fraction(1, 3)
y = Fraction(4, 6)

print(x, y)

> 1/3 2/3

x

> Fraction(1, 3)

y

> Fraction(2, 3)
```

From one floating-point number:

```python
from fractions import Fraction

x = Fraction(.25)
y = Fraction(1.8)

print(x)

> 1/4

print(y)

> 8106479329266893/4503599627370496
```

Using floating-point number **.as_integer_ratio()** method:

```python
t = (1.25).as_integer_ratio()

print(t)

> (5, 4)

from fractions import Fraction

f = Fraction(*t)

print(f)

> 5/4
```

And back:

```python
print(float(f))

> 1.25
```

Using fractions **.from_float()** method:

```python
from fractions import Fraction

f = Fraction.from_float(1.25)

print(f)

> 5/4
```

#### 1.2.2. How operations work?

```python
from fractions import Fraction

x = Fraction(1, 4)
y = Fraction(1.25)

print(x + y)
> 3/2

print(x - y)
> -1

print(x * y)
> 5/16
```

```python
from fractions import Fraction

x = Fraction(1, 10) + Fraction(1, 10) + Fraction(1, 10) - Fraction(3, 10)

x

> Fraction(0, 1)
```

#### 1.2.3. How to simplify fractions?

```python

x = 22 / 7

print(x)

> 3.142857142857143

from fractions import Fraction

x = Fraction(3.142857142857143)

x

> Fraction(7077085128725065, 2251799813685248)

x = x.limit_denominator(10)

print(x)

> 22/7
```
